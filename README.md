## 🏗️ Lab Environment

Documentation and configurations for the **CEH Ultimate Home Lab**, optimized for hardware-level performance and isolated network testing.

* **Primary OS:** Kali Linux (Bare Metal Optimization)
* **Performance:** Custom `zRAM` deployment (75% RAM allocation) for SSD longevity and heavy multitasking.
* **Virtualization:** VMware Workstation Pro & GNS3 integration.
* **Network Simulation:** Cisco Packet Tracer featuring advanced VLSM & Router-on-a-Stick topologies.
* **Containerization:** Docker with MacVLAN isolation (e.g., OWASP Juice Shop).
